Installation von node.js

Laden Sie Node.js von nodejs.dev runter.

Installation von GitHub Desktop

Laden Sie sich GitHub Desktop von GitHub runter.

Detaillierte Installation von node.js

Schritt 1: Node.js herunterladen

Öffnen Sie Ihren Webbrowser und besuchen Sie die offizielle Node.js-Website.

URL: https://nodejs.org/

Klicken Sie auf den "LTS"-Download-Button, um die empfohlene stabile 64-bit Version herunterzuladen.

Schritt 2: Installation starten

Nachdem der Download abgeschlossen ist, öffnen Sie die heruntergeladene Installationsdatei (z.B., "node-vxx.x.x-x64.msi").

Wenn Sie eine Benutzerkontensteuerung (UAC)-Warnung erhalten, bestätigen Sie, dass Sie die Änderungen zulassen möchten.

Schritt 3: Installationsassistent

Der Node.js-Installationsassistent wird geöffnet. Klicken Sie auf "Next" (Weiter).

Lesen Sie die Lizenzvereinbarung und akzeptieren Sie sie. Klicken Sie erneut auf "Next".

Wählen Sie das Installationsverzeichnis aus (standardmäßig ist dies in Ordnung). Klicken Sie auf "Next".

Wählen Sie die gewünschten Komponenten aus (normalerweise können Sie die Standardeinstellungen belassen) und klicken Sie auf "Next".

Wählen Sie das Startmenü-Ordner oder lassen Sie die Standardeinstellungen unverändert. Klicken Sie auf "Next".

Wählen Sie die Option "Automatisch" für die PATH-Variable, damit Node.js global verfügbar ist. Klicken Sie auf "Next".

Klicken Sie auf "Install" (Installieren), um die Installation zu starten.

Schritt 4: Installation abschließen

Der Installationsassistent wird den Vorgang abschließen. Klicken Sie auf "Finish" (Fertigstellen), um die Installation abzuschließen.

Schritt 5: Überprüfen der Installation

Öffnen Sie die Eingabeaufforderung (CMD) und geben Sie den folgenden Befehl ein:

node -v

Sie sollten die installierte Node.js-Version angezeigt bekommen.

Geben Sie auch den folgenden Befehl ein, um die npm-Version (Node Package Manager) zu überprüfen:

npm -v
